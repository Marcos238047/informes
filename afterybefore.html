After y Before.

Ideas principales

  • Pseudeoelementos after y before

    los pseudoelementos after y before son pseudoelementos que insertan elementos de contenido desde el css en lugar de modificar elementos html ya existentes, los cuales pueden ser:

    • Una cadena de caracteres

      • utilizando el valor "content: "text"" podemos untroducir cadenas de caracteres desde css

    • una imagen

      • utilizando el valor "content:"URL(ejemplo.jpg)""

    • nada
  • valor "content: "" "

    • dejando vacio el valor de content podemos crear formas y figuras dese css, utilizando formas geometricas preestablecidas.

apuntes de la clase

Usualmente se pueden crear la mayoria de formas utilizando 3 o menos formas geometricas para la misma figura.

Resumen

Los pseudoelementos after y before nos permiten crear elementos que modifiquen la apariencia de nuestro proyecto sin la necesidad de incluirlos entre los elementos html normales, realmente utiles para modificar aspectos de una pagina web sin tener que modificar realmente su html o el css d eotro elemento, permitiendo arreglos faciles y rapidos.

utilizando el valor content: "" podemos utilizar diferentes formas geometricas para crear elementos visuales que no formen parte de una imagen y que podamos modificar directamente utilizando sus propiedades css, lo que, utilizado de manera inteligente, puede otorgarnos un control enorme sobre las formas que querramos insertar.
