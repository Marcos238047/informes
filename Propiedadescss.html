Propiedades de CSS3

Ideas principales

  • Las diferentes propiedades de Css 3:
    • border radious

      • Propiedad que redondea las esquinas del Border

    • box shadow

      • Propiedad que le da una sombra a una seccion

    • text shadow

      • Propiedad que le da una sombra a un texto

    • letter spacing

      • propiedad que modifica el espacio entre las letras de un texto

    • font face

      • propiedad que cambia la fuente de un texto a una local

    • radial gradient

      • Propiedad de CSS que crea un gradiente de color desde el centro hacia fuera

    • linear gradient

      • Propiedad de CSS que crea un gradiente de color de un lado a otro

    • outline offset

      • propiedad que aleja el borde del objeto principal

    • border image

      • propiedad que otorga un borde a una imagen

    • transform y sus subetiquetas:
      • Rotate

        • Propiedad que Rota el contenido de un elemento

      • Scale

        • Propiedad que cambia el tama√±o del contenido de un elemento

      • Skew

        • Propiedad que Deforma el contenido de un elemento

      • Translate

        • Propiedad que Mueve el contenido de un elemento fuera de su posicion real

Apuntes de la clase

Puedes aplicarle multiples Propiedades a un mismo elemento para modificar mucho su apariencia, pero no deberias hacerlo solo por hacerlo, intenta mantener la estructura de un sitio.

Resumen

El CSS 3 otorga al usuario la acapacidad de otorgar ciertas propiedades especiales a los elementos html para modificar su apariencia y posicion sin tener que aplicar una multitud de reglas al elemento. las diferentes propiedades pueden aplicar efectos que son utiles para el usuario.
