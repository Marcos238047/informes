El header en la web.

Ideas principales

  • La importancia del header en la web

    • el header es uno de los elementos genericos de una web el cual es casi siempre de la misma manera, es un elemento muy util para crear una buena estructura web

  • Reglas de un header

    • los headers son siempre elementos que van en la parte superior de la pagina que presentan los elemenos importantes para la navegacion, lo que crea un sentimiento de orden que empuja al usuario a navegar, ademas, el usuario siempre debe llevar el logo.

  • Elementos que puede incluir un header

    un heaer puede contener muchos elementos, pero los que usualmente se utilizan los siguientes

    • El logo
    • barras de navegacion
    • buscadores
    • datos de la compañia
    • redes sociales de la compañoa
    • tiendas online
    • botones destacados
    • barras fijas
    • selector de idioma
    • formularios de suscripcion
    • varios otros

apuntes de la clase

Los elementos de un header comunmente deben mantenerse en cantidad baja para crear una vision clara del sitio.

Resumen

El header es una de las partes más importantes de una página web, es el elemento que los usuarios visualizan primero cuando llegan al proyecto, por lo que la cabecera de la web debe ser ordenada y limpia para que pueda ser visualizada con un solo vistazo y no de un lugar de confusión.
Debemos tener en cuenta que el header es un espacio genérico que tendrá el mismo diseño en todas las páginas de un sitio web.
Lo primero que se ebe hacer es un listado con todos los elementos a incluir dentro del header de tu web de forma obligatoria, por motivos de branding, navegacion u otros
Los elementos del header dependen del objetivo del sitio, por lo que va a bariar de sitio a sitio.
