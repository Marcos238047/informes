Resumen

Los formularios de una pagina w4eb son elementos que permiten a los usuarios enviar informacion a la pagina, usualmente para recibir una respuesta.

comunmente se utilizan archivos php en el lado del servidor para que reciban los valores que el dev le pidio al usuario y los guardan en una base de datos desde los que se pueden consultar

tambien se permite a los elementos de un form tener propiedades css, la gran mayoria de las etiquetas aplicables a otros elementos html estan disponibles pero se debe tener cuidado especificando cuales de los elementos seran los que han de ser modificados, ya que se pueden crear modificaciones no deseadas a elementos diferentes si no se utliza la especifidad adecuada.
