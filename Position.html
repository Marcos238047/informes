Position

Ideas principales

  • Etiqueta position

    • la etiqueta position es una etiqueta que especifica el posicionamiento para un elemento html y puede ser clasificada como:

    • Static

      • Es la posicion default, el objeto se posiciona con el flow regular de la pagina

    • Relative

      • Se posiciona en relacion al contenedor y sus esquinas e ignora los demas elementos

    • Fixed

      • Se posiciona en relacion a la pantalla, es decir, si se scrollea el elemento se quedara en su sitio

    • Absolute

      • El objeto se posiciona se posicione en relacion a algun ancestro, pero se queda en la pantalla anque se scrolle

    • Sticky

      • Es un objeto posicionado en base a la posiciond e scroll del usuario y despues de entcontrarlo se quedara aunque se scrollee pasado al mismo

  • Z-Index

    • por medio de CSS los elementos con posicion relativa se pueden acomodar por encima o debajo de otros dependiendo de su posicion, a√±adiendo tambien la profundidad como posicion

Resumen

La etiqueta position es una manera facil de modificar la manera en la que un documento esta estructurado visualmente y ayuda facilmente a crear paginas con elementos dinamicos que estan por encima o debajo de los elementos.

La etiqueta display puede hacer extremadamente facil hacer elementos de menu que siempre se queden en pantalla.
