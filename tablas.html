Resumen

Los elementos table se tratan de tablas de html, que permiten organizar y mostrar informacion como una de ellas utilizando las etiquetas TR (table row) para crear filas y TD (table column) para crear columnas y asi especificar la forma y las casillas de tu tabla.

las tablas estan sujetas a las etiquetas regulares para los elementos css, asi como lo estan sus contenidos, solo se debe ser especifico con el elemento para poder seleccionarlo sin afectar el resto de la tabla, utilizando elementos de especifidad como el nth child u otros.

existen algunos elementos CSS solo disponibles para tablas como Rowspan y Colspan, las cuales aumentan la cantidad de filas y columnas que una casilla abarca respectivamente, border spacing para cambiar los bordes de las celdas en relacion con la tabla y muchas otras etiquetas.
