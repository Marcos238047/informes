Display

Ideas principales

  • Etiqueta display

    • Una etiqueta que modifica como se muestra un elemento en una pagina

    • Inline

      • Hace que un elemento se alinie dentro de la misma linea

    • Block

      • El default, los elementos se posicionan tomando todo el espacio en el axis X que pueden y un nuevo elemento se coloca en una nueva linea

    • Inline-block

      • El elemento se formatea como inline, pero al que se le pueden aplicar altura y anchura

    • Flex

      • Parecido al inline, pero se le pueden aplicar diferentes sub etiquetas que modifican la manera en la que los contenidos del elemento se muestran

    • None

      • El elemento no se muestra, se usa para esconder un elemento

Apuntes de la clase

Los elementos puedes aplicar diferentes displays a elementos dentro de un solo contenedor.

Resumen

El display se refiere a la manera en la que un elemento html es presentado visualmente en una pagina web,se puede utlizar para estructurizar facilmente un sitio web.
